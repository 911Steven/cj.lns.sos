技术特点：
- 像eclipse的插件机制一样，sos也是由一系统插件构成，其插件体系还支持父子结构
- 五层分布式网络节点架构：从展现、中间路由（两级）、服务BUS，数据库
- 自2015年上线，在2018年7月下线
