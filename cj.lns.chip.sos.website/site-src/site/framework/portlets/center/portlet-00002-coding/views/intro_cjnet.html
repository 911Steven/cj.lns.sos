cjnet 调试、测试命令行工具

  • 作者：cj (赵向彬)发布日期：2016年8月19日 18:40
关键词：调试工具 net cjnet
摘要：用于调试神经元应用和netsite应用，模仿客户端发起请求调用。
下载cjnet如欲使用最新版本且到开发平台中下载
正文： 
一、cjnet与远程服务器（可能是神经元的net也可能是netsite的net)的连接

CJdeMacBook-Pro:cjnet carocean$ sh cjnet.sh -m

usage: cjnet
 -d,--debug           调试命令行程序集时使用，需指定以下jar包所在目录
                           cj.studio.ecm.chip.cjnet
 -h,--host            必须指定远程地址，格式：-h ip:port
 -l,--log                  充许网络日志输出到控制台
 -m,--man                  帮助
 -P        设置客户端连接属性,格式为：-Pproperty=value
 -t,--tranprotocol    指定传输协议，支持：
                           udt,tcp,http,websocket,udp,local
 -T    设置客户端连接池属性，
                           工作线程数-Twork=n,池最大大小-Tmax=n,
                           池空间数-Tidle=n,
                           池最小数-Tmin=n,
                           超时时间-Ttimeout=n
                           
 连接远程神经元：8085 rio-tcp服务器 
 CJdeMacBook-Pro:cjnet carocean$ sh cjnet.sh -h localhost:8085 -t rio-tcp

二、请求侦的发送

localhost:8085 >get -m
usage: get
 -c,--content           侦的内容。
 -C     设置回路的头,格式为：-CHheadName=value
 -H     设置侦的头，格式：
                        -Hheadname=value
 -m,--man               帮助
 -P    设置侦的参数,格式为：-Pparamname=value
 -s,--sync              指定为同步发送消息.默认是异步发送。
                        如果是同步，消息将在命令窗口中交替显示，
                        如果是异步，消息在到达时随机显示
 -t,--protocol     侦的请求头包含的协议。
 -u,--url          侦的请求头包含的url。
 
 发送请求：
 localhost:8086 >get -u / -t http/1.1 -Hcj-circuit-sync=true
